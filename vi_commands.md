# vi commands
```
i --> insert mode
esc --> escape mode/command mode
:wq --> save and exit
:x --> same as above
:q! --> exit w/o saving
:q --> quit vi
esc mode + dd --> entire line delete
esc mode + yy --> copy line, called yank
esc mode + p --> paste line
esc mode + I --> line beginning
esc mode + A --> line end
esc mode + shift+d --> cut from the cursor to the end of the current line
                 p ---> to paste the same line
esc mode + shift+c --> cut from the cursor to the end of the current line and insert mode is on
esc u --> to undo recently pasted text
to paste any clipboard copied text, go to insert mode and ctrl+v directly
```
